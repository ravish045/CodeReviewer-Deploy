# 🚀 Code Reviewer (AI Powered)

A full-stack SaaS application that reviews code using AI and provides optimized solutions with detailed feedback.

## 🌐 Live Demo

https://code-reviewer-deploy.vercel.app/

## ⚡ Features

* AI-based code review (Llama 3.1 via OpenRouter)
* Multi-language support (C, C++, Java, Python, JS)
* Google OAuth login
* Guest mode (no login required)
* Review history (stored in MongoDB)
* Modern UI with animations

## 🛠 Tech Stack

Frontend: React + Vite + Tailwind
Backend: Node.js + Express
Database: MongoDB
Auth: JWT + Google OAuth
AI: OpenRouter (Llama 3.1)

## 📦 Deployment

Frontend: Vercel
Backend: Render

## 🧠 What I Learned

* Full-stack deployment
* Authentication systems
* API integration (LLM)
* Production debugging
